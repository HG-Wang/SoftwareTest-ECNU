练习1：ISP建模与组合标准

题目描述

对三角形分类方法 triang(int a, int b, int c) 进行完整的ISP分析。

public String triang(int a, int b, int c) {
    if (a <= 0 || b <= 0 || c <= 0) return "invalid";
    if (a == b && b == c) return "equilateral";
    if (a == b || b == c || a == c) return "isosceles";
    if (a + b > c && b + c > a && a + c > b) return "scalene";
    return "invalid";
}Java

分析步骤

可测试功能：triang(int a, int b, int c)，输入三个整数，返回三角形类型字符串。

参数：a (int), b (int), c (int)

每个参数与0的关系：{>0}, {=0}, {<0}

精化：{>1}, {=1}, {=0}, {<0}

ACoC: 4×4×4 = 64个测试

ECC: max(4,4,4) = 4个测试

PWC: ≥ 4×4 = 16个测试

BCC: 1+3+3+3 = 10个测试

ECC示例（4个测试）：

  • (2, 2, 2) — 所有>1
  • (1, 1, 1) — 所有=1
  • (0, 0, 0) — 所有=0
  • (-1, -1, -1) — 所有<0

练习2：组合标准计算

题目描述

给定以下特征和块数，计算各组合标准的测试数：

特征块数块定义
F13{小}, {中}, {大}
F22{有效}, {无效}
F34{A}, {B}, {C}, {D}
F42{是}, {否}

计算

ACoC = 3 × 2 × 4 × 2 = 48 个测试

ECC = max(3, 2, 4, 2) = 4 个测试

PWC ≥ 4 × 3 = 12 个测试（两个最大特征F3和F1的块数之积）

假设基本块为每特征第一个块。

BCC = 1 + (3-1) + (2-1) + (4-1) + (2-1) = 1 + 2 + 1 + 3 + 1 = 8 个测试

练习3：边界值分析

题目1：电梯过载

电梯最大承重280kg，使用2-value和3-value BVA设计测试用例。

等价类：{≤280}（正常）, {>280}（超载）

边界值：280（域内边界）, 281（域外边界）

测试值：279, 280, 281（共3个）

边界值280：279, 280, 281

测试值：279, 280, 281（共3个，边界本身已在集合中）

题目2：年份输入

有效年份范围1900-2020，使用3-value BVA设计测试用例。

边界值：1900, 2020

对1900：1899, 1900, 1901

对2020：2019, 2020, 2021

测试值：1899, 1900, 1901, 2019, 2020, 2021（共6个）

题目3：购物折扣

购物金额分档：0-100无折扣，101-500九折，501+八折。使用BVA设计测试用例。

边界值：0, 100, 101, 500, 501

2-value测试值：-1, 0, 100, 101, 500, 501（共6个）

3-value测试值：-1, 0, 1, 99, 100, 101, 102, 499, 500, 501, 502（共11个）

题目4：PIN验证

PIN为4位数字（0000-9999），长度必须恰好为4位。使用BVA设计测试用例。

等价类：{长度<4}, {长度=4}, {长度>4}

边界值（长度）：4

测试值：3位PIN、4位PIN、5位PIN

具体：123, 1234, 12345（或000, 0000, 00000）

练习4：ISTQB概念辨析

选择题

A. 组件测试   B. 系统测试   C. 回归测试   D. 验收测试

答案：C — 回归测试是测试类型，不是测试级别。

A. 语句覆盖   B. 等价类划分   C. 路径覆盖   D. 数据流覆盖

答案：B — 等价类划分基于输入输出，不考虑内部结构。其他选项都是基于代码结构的白盒技术。

A. 非正式审查   B. 演练   C. 技术审查   D. 检查（Inspection）

答案：D — 检查是最正式的审查类型，有严格的角色分工、流程和度量。

A. 将测试推迟到开发后期   B. 尽早开始测试活动

C. 只做单元测试   D. 不需要验收测试

答案：B — 左移强调在SDLC的早期阶段就开始测试活动，以降低缺陷修复成本。

A. 块之间没有交集   B. 块的并集覆盖整个输入域

C. 每个块至少有一个测试   D. 每个特征有多个块

答案：A — 互斥性要求 bi ∩ bj = ∅, ∀i≠j，即任意两个不同的块之间没有交集。
