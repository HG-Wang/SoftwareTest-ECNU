练习1：控制流图覆盖分析

题目描述

给定以下Java方法，画出控制流图，并分别给出满足边覆盖(EC)、边对覆盖(EPC)和主路径覆盖(PPC)的测试需求和测试路径。

public int findMax(int[] arr) {
    int max = arr[0];          // 1
    for (int i = 1; i < arr.length; i++) {  // 2
        if (arr[i] > max) {    // 3
            max = arr[i];      // 4
        }
    }
    return max;                // 5
}Java

分析步骤

节点1(初始化), 节点2(循环判断), 节点3(条件判断), 节点4(更新max), 节点5(返回)。边: 1→2, 2→3, 2→5, 3→4, 3→2, 4→2。

TR = {(1,2), (2,3), (2,5), (3,4), (3,2), (4,2)}。测试路径需要覆盖所有边。

TR包含所有长度≤2的路径对。

找出所有简单路径，识别主路径。

练习2：数据流覆盖分析

题目描述

对于以下代码，标识所有def和use集合，列出DU对，并给出满足All-Uses覆盖(AUC)的测试路径。

public boolean isSorted(int[] a) {
    boolean sorted = true;        // 1
    for (int i = 0; i < a.length - 1; i++) {  // 2
        if (a[i] > a[i+1]) {      // 3
            sorted = false;       // 4
            break;
        }
    }
    return sorted;                // 5
}Java

分析步骤

def(1)={sorted}, def(2)={i}, def(4)={sorted}

use(2)={i, a}, use(3)={a, i, sorted}, use(4)={sorted}, use(5)={sorted}

(1,3,sorted), (1,5,sorted), (4,5,sorted), (2,3,i), (2,3,a), ...

需要覆盖每个def到每个use。
