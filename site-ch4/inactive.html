非活动子句覆盖 (Inactive Clause Coverage)

ICC 系列覆盖标准通过控制非活动子句的取值来保证子句对谓词的独立影响力，是 MC/DC 的理论基础。

§ 8.1.3 ICC / GICC / RICC MC/DC

ICC 定义#

Inactive Clause Coverage (ICC) 的核心思想是：对于谓词 p 中的每个子句 ci，当 ci 为 非活动子句（即不决定谓词值）时，需要满足以下四个要求：

#要求说明
1ci = T, p = T子句为真，谓词也为真
2ci = F, p = T子句为假，谓词仍为真
3ci = T, p = F子句为真，谓词为假
4ci = F, p = F子句为假，谓词也为假
ICC 保证了子句在非活动状态下，所有 (ci, p) 的组合都被覆盖，从而确保谓词覆盖 (PC) 被隐含满足。

GICC 与 RICC#

GICC (General ICC)

  • 弱化版本的 ICC
  • 对每个子句 ci，当其为非活动时，需覆盖 p=T 和 p=F 各至少一次
  • 不要求其他子句取值相同
  • 测试用例数量更少

RICC (Restricted ICC)

  • 严格版本的 ICC
  • 对每个子句 ci，当其为非活动时，需覆盖全部 4 种 (ci, p) 组合
  • 要求其他子句（minor clauses）取值相同
  • 测试用例数量更多，但覆盖更充分

ICC 隐含谓词覆盖#

ICC 始终保证 Predicate Coverage (PC)。因为在 ICC 的 4 个要求中，已经包含了 p=T 和 p=F 的情况，无论子句是活动还是非活动状态，谓词的真假值都会被覆盖到。

定理：ICC ⊇ PC。任何满足 ICC 的测试集必然满足 Predicate Coverage。

不可行性示例#

考虑谓词：

p = (a > b) ∧ (b > c) ∨ c > a

该谓词有三个子句：

  • c1: a > b
  • c2: b > c
  • c3: c > a
当三个子句同时为 True 时：a > b > c > a，这形成了一个矛盾循环，不可能存在满足条件的输入值。这意味着某些 ICC 要求可能是不可行的 (infeasible)。

GACC 不能保证 PC 的示例#

考虑异或谓词 p = a ⊕ b，其真值表如下：

行abp = a ⊕ b
1TTF
2TFT
3FTT
4FFF

GACC 要求：对每个子句，存在使谓词为 T 和 F 的取值。选择 行 {1, 4} 或 行 {2, 3}：

  • 行 {1, 4}：a=T→p=F, a=F→p=F, b=T→p=F, b=F→p=F — 满足 GACC 但 p 从未为 T，不满足 PC
  • 行 {2, 3}：类似地，满足 GACC 但 p 从未为 F，不满足 PC
GACC 在异或 (XOR) 谓词上不能保证 Predicate Coverage，因为它不要求其他子句取值一致。

使子句决定谓词#

Definitional 方法

对于复合谓词，我们可以通过定义式分解来确定每个子句是否决定谓词：

XOR 的定义：p = (a ∧ ¬b) ∨ (¬a ∧ b)

子句 a 决定 p 当 b 固定时：p = a ⊕ b，改变 a 一定改变 p，所以 a 总是决定 p。

a 决定 p 当 b = F 时（此时 p = a）。

b 决定 p 当 a = F 时（此时 p = b）。

a 决定 p 当 b = T 时（此时 p = a）。

b 决定 p 当 a = T 时（此时 p = b）。

a 决定 p 当 b ∧ c = F 时。

b 决定 p 当 a = F 且 c = T 时。

c 决定 p 当 a = F 且 b = T 时。

Tabular 方法

使用真值表的行对 (row pairs) 来判断子句决定关系：如果仅改变子句 ci 的值会导致谓词 p 改变，则 ci 在该行决定 p。

abp = a ∨ ba 决定 p?b 决定 p?
TTT否 (改 a→仍 T)否 (改 b→仍 T)
TFT否 (改 a→F, 但需看对)是 (改 b→T, 对应行1)
FTT是 (改 a→T, 对应行1)否 (改 b→F, 对应行4)
FFF是 (改 a→T, 对应行2)是 (改 b→T, 对应行3)

MC/DC 覆盖标准#

Modified Condition/Decision Coverage (MC/DC) 要求：

  • 每个条件（子句）都独立地影响判定（谓词）的结果
  • 每个条件取 T 和 F 至少各一次
  • 每个判定取 T 和 F 至少各一次

测试数量：MC/DC 通常需要约 N + 1 个测试用例（N 为条件数），相比组合覆盖的 2N 大幅减少。

MC/DC 是航空电子标准 DO-178B/C Level A 的强制要求，在工业实践中被广泛采用。
